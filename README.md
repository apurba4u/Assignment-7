# 🌿 KeenKeeper

## 📌 Project Description

KeenKeeper is a relationship management web application that helps users track and maintain meaningful connections with friends, family, and professional contacts. It allows users to monitor interaction frequency, set communication goals, and visualize engagement through analytics.

---

## 🚀 Technologies Used

* ⚛️ React (Next.js App Router)
* 🎨 Tailwind CSS + DaisyUI
* 📊 Recharts (for data visualization)
* 💾 LocalStorage (for interaction tracking)
* 🔔 React Toastify (for notifications)
* 🎯 Lucide React Icons

---

## ✨ Key Features

### 1. 📇 Smart Contact Management

* Add and manage multiple contacts
* Track relationship notes and preferred communication methods
* Status indicators (Overdue, Almost Due, On Track)

---

### 2. 📊 Interaction Tracking & Analytics

* Log interactions (Call, Text, Video)
* Automatic counting and visualization using charts
* Timeline view of all activities

---

### 3. ⏰ Goal-Based Relationship Tracking

* Set communication goals for each contact
* Track days since last interaction
* Get reminders for upcoming or overdue connections
